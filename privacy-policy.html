Privacy Policy for Wynnbuilder Mana Calculator

Effective Date: April 17, 2025

The Wynnbuilder Mana Calculator ("the Extension") is a Chrome extension designed to add a graphical user interface (GUI) to calculate mana consumption on Wynnbuilder pages (https://wynnbuilder.github.io/). This Privacy Policy explains how we handle user data in compliance with Google Chrome Web Store policies.

1. Data We Collect

The Extension collects the following data to perform mana calculations:

  • Website Content: Mana regeneration stats, spell costs, total mana pool, total HP, and spell names displayed on Wynnbuilder pages.
  • User Activity: User inputs in the GUI, such as spell cycle text, clicks per second (CPS), slider values (e.g., enemy hit count, totem count), button toggles (e.g., Blood Pact, Arcane Transfer), and mouse interactions for dragging the GUI.

No personally identifiable information, health information, financial data, authentication data, personal communications, location, or web history is collected.

2. How We Use Your Data

The collected data is used solely to:

  • Calculate mana and HP consumption and display results in the Extension’s GUI.
  • Update calculations based on user inputs (e.g., spell cycle, CPS, toggles).
  • Allow repositioning of the GUI through drag-and-drop interactions.

All data processing occurs locally on your device within the Chrome browser. The Extension does not store, transmit, or share any data with external servers or third parties.

3. Data Sharing and Disclosure

We do not sell, transfer, or share user data with any third parties. The Extension operates entirely within the Chrome browser and does not connect to external services.

4. Data Storage

The Extension does not store any user data locally or remotely. All data is processed in real-time during your interaction with Wynnbuilder pages and is discarded when you close the browser tab.

5. Security

We are committed to protecting user data. Since the Extension processes data locally and does not transmit it, there is minimal risk of data exposure. We adhere to Chrome’s data minimization principles by collecting only the data necessary for the Extension’s functionality.

6. Your Rights

As the Extension does not collect personally identifiable information or store data, there are no user accounts or data to manage. You can uninstall the Extension at any time via Chrome’s extension settings.

7. Changes to This Privacy Policy

We may update this Privacy Policy to reflect changes in the Extension’s functionality or Chrome Web Store policies. The latest version will be available at this URL, with the effective date updated accordingly.
